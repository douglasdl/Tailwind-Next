# Tailwind-Next

Create the project:
```sh
npm create next-app tailwind-next
```

Install the dependencies:
```sh
npm i lucide-react
```

Install the development dependencies:
```sh
npm i @rocketseat/eslint-config -D
npm i prettier-plugin-tailwindcss -D 
```

Start the project:
```sh
npm run dev
```
